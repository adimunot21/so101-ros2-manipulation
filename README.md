# SO-101 ROS2 Manipulation Stack

A production-quality ROS2 manipulation stack for the SO-101 robot arm —
integrating perception, motion planning, control, and imitation learning —
in MuJoCo simulation with sim-to-real transfer capability.

## Status

🚧 **Under active development** — Phase 0 complete.

## Quick Start
```bash
# ROS2 + ML environment
ros2ml

# LeRobot training environment
conda activate lerobotROS
```
